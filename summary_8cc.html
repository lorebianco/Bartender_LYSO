void Bartender_Summary ( Double_t duration,
Bar * bar,
SiPM * sipm )

Writes the summary of the Bartender to a text file, updating it at the end of every simulation. The summary includes:

  • Bartender ID (equal to the ID of the MC_LYSO), date, username and duration;
  • Settings related to the MPPC working point.
Parameters
durationThe duration (in seconds) of the simulation
barThe Bar class instance pointer
sipmThe SiPM struct instance pointer
